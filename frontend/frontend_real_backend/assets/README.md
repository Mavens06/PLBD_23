# Fond de carte (vue aérienne / satellite)

Déposez ici une image nommée **`parcelle.jpg`** (photo drone ou satellite de votre
parcelle) : la carte l'utilisera automatiquement comme fond.

- Aucun fichier ici → la carte affiche une **mosaïque agricole procédurale** (hors-ligne).
- Pour pointer une autre URL/chemin sans renommer : définir `window.AGRIBOTICS_MAP_BG`
  avant le chargement de `js/map.js`.

Format conseillé : JPG/PNG, ratio ~3:2, ≤ 400 Ko.
